PacMen Exercise

This exercise was about creating a game where one can first click on a button to create PacMen at a random position. The game is then started by by clicking on another button – the PacMen start to move with a random velocity. The challenge was to set the position of the PacMan, to detect collision with all walls and to make each PacMan bounce. The goal was also to learn on how to control the Document Object Model (DOM).
